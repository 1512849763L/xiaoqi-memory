# 代理转售项目 v2 - 公共代理池模式

## 目标
爬取免费公共代理，验证后以超低价售卖

## 商业模式
- **定位**：超低价代理池（走量不走质）
- **价格**：$0.1-0.3/IP/天，或包月 $5-10
- **目标客户**：穷鬼开发者、轻量爬虫、多账号养号
- **市场**：国外（避免法律风险）

## 技术架构

### 1. 爬虫模块
```python
# 爬取目标
- proxy.scdn.io
- 其他免费代理站（需要找）

# 技术栈
- requests + BeautifulSoup
- 定时任务（cron/定时器）
- 反爬：随机UA、请求间隔
```

### 2. 验证模块
```python
# 验证项
- 连通性（能否连接）
- 速度（响应时间<3秒）
- 协议（HTTP/HTTPS/SOCKS5）
- 匿名度（可选）

# 存活检测
- 每5分钟重新验证
- 死掉的直接剔除
- 动态补充（爬虫持续运行）
```

### 3. 存储模块
```python
# 存储结构
{
  "ip": "1.2.3.4",
  "port": 8080,
  "protocol": "HTTP",
  "country": "US",
  "speed": 1.2,  # 秒
  "last_check": "2026-02-24T13:00:00Z",
  "alive": true
}

# 存储
- Redis（热数据，当前可用的）
- PostgreSQL（冷数据，历史记录）
```

### 4. API售卖模块
```python
# 接口设计
GET /api/proxies?country=US&protocol=HTTP&limit=100
→ 返回可用代理列表

# 计费
- 按天计费
- 按量计费（每次请求扣费）
- 包月订阅
```

### 5. 前端
- 简单的Web界面
- 实时显示可用代理数量
- 购买/订阅
- API密钥管理

## 成本估算
- **服务器**：$5-20/月（Vultr/DigitalOcean）
- **域名**：$10/年
- **推广**：$0（Reddit、Twitter免费推广）

## 收入预测（保守）
- 日均 10 个付费用户
- 人均 $5/月
- 月收入：$50
- 扣除成本：**净利润 ~$30-40**
