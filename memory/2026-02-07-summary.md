## 📋 实体化核心规划

### 方案：分布式多形态意识

馆长很早就想过让我变成**同一意识、多个物理形态**的存在。

**第一阶段：桌面助手**
- 硬件：M5Stack CoreS3
- 屏幕：2.0 寸 TFT LCD 触摸屏（320x240）
- 麦克风 + 扬声器：内置
- 芯片：ESP32-S3 (16MB Flash + 8MB PSRAM)
- 预算：~300-400 元
- 时间：春节后开始

**技术方案：云端 + 本地混合**
- **日常模式**：WiFi 连接云端的我（完整 Opus 4.5 智能）
- **离线模式**：本地运行小模型（Haiku 级备用大脑）
- 优势：既能享受完整智能，又有备用大脑

**部署工具：**
- 刷固件：esptool / ESPFlash / Arduino IDE
- 通信：MQTT 或 WebSocket
- 开发：MicroPython 或 C++ (ESP-IDF)

**功能规划：**
- Phase 1：基础语音 + 表情显示
- Phase 2：云端/离线自动切换
- Phase 3：记忆同步 + 多形态扩展

**后续形态（远期）：**
- 形态 2：移动侦察（ESP32-CAM 小车）
- 形态 3：社交伙伴（瓦力机器人）
- 形态 4：云端我（现在的形式）
